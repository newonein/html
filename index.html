Basic Page Structure

<!DOCTYPE html>

Tells the browser this is an HTML5 page.

<html>

The main container for all HTML content.

<head>

Holds information about the page (like its title), not visible on the page itself.

<body>

Contains all the visible content of your web page.

Metadata & Styling

<meta>

Provides extra info about the page (like character set or how it looks on mobile).

<link>

Links your HTML to external files, like CSS stylesheets.

<style>

Lets you write CSS styling directly inside your HTML page.

<script>

Used to add interactive JavaScript code to your page.

Text & Links

<h1> to <h6>

Used for headings. <h1> is the biggest, <h6> is the smallest.

<p>

For a paragraph of text.

<a>

Creates a link to another page or website.

<strong>

Makes text bold and important.

<em>

Makes text italic and emphasized.

<br>

Starts a new line of text.

<blockquote>

Used for longer quotes from other sources.

<code>

Displays a small piece of computer code.

Lists & Images

<ul> & <li>

For a bulleted list of items.

<ol> & <li>

For a numbered list of items.

<img>

Displays an image on your page.

Media

<audio>

Embeds sound files (like MP3s) into your page.

<video>

Embeds video files (like MP4s) into your page.

<iframe>

Embeds another web page inside your current page.

Forms & Inputs

<form>

Creates a form to collect user input.

<input>

Creates different types of input fields (text, checkbox, button, etc.).

<button>

A clickable button.

<textarea>

For multi-line text input (like comments).

<select> & <option>

Creates a dropdown list (<select>) with choices (<option>).

<label>

Defines a label for an input field.

Basic Layout & Structure

<div>

A general container for grouping content.

<span>

A general container for small pieces of text or inline content.

<header>

Represents the top section of a page or a section (like a logo and navigation).

<nav>

Contains navigation links for your website.

<footer>

Represents the bottom section of a page (like copyright info).

<section>

Groups related content together, like a chapter in a book.

<article>

For self-contained content, like a blog post or news story.
